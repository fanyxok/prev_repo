## `bash_proxy_wsl2.sh` 
usage:
- `. ./bash_proxy_wsl2.sh set`, both set wsl2 and wls2 git
- `. ./bash_proxy_wsl2.sh unset`, both unset wsl2 and wls2 git
- `. ./bash_proxy_wsl2.sh test`, test and print current proxy address
